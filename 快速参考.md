## 🔐 测试模式

```powershell
# 启用（需要重启）
bcdedit /set testsigning on
shutdown /r /t 0

# 禁用（需要重启）
bcdedit /set testsigning off
shutdown /r /t 0

# 检查状态
bcdedit /enum | findstr testsigning
```

## 🛠️ 常用命令

```powershell
# 停止服务
sc stop SbieSvc

# 启动服务
sc start SbieSvc

# 查看服务状态
sc query SbieSvc

# 重新安装驱动
sc delete SbieDrv
sc create SbieDrv type= kernel start= demand binPath= "C:\Program Files\Sandboxie-Plus\SbieDrv.sys"

# 查看签名
signtool verify /v /pa "C:\Program Files\Sandboxie-Plus\SbieDrv.sys"
```

## ✅ 验证解锁

打开 **Sandboxie Control** → 右键沙盒 → **沙盒设置**

检查以下功能是否可用（无需证书提示）：

| 功能 | 位置 |
|------|------|
| ✅ 高级安全模式 | 安全 → UseSecurityMode |
| ✅ 系统调用锁定 | 安全 → SysCallLockDown |
| ✅ 加密沙盒 | 资源访问 → ConfidentialBox |
| ✅ DNS过滤 | 网络 → NetworkDnsFilter |
| ✅ 独立桌面 | 外观 → UseSandboxDesktop |

## 🐛 快速故障排除

| 问题 | 解决方案 |
|------|---------|
| 编译失败 | 安装 WDK 扩展，重启 VS |
| 驱动加载失败 | 启用测试模式并重启 |
| 服务无法启动 | 检查事件查看器 |
| 功能仍需证书 | 确认新文件已复制，重启服务 |
| 蓝屏 | 进入安全模式，运行 restore.bat |
